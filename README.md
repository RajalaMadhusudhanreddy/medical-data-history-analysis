# 🏥 Medical Data History Analysis

## 📌 About
Analysis of 1000 patient medical records using MySQL 
to identify health patterns, obesity trends, 
and patient demographics.

## 🛠️ Tools Used
- MySQL / MySQL Workbench

## 📊 Key Findings
- Analyzed 1000 patient records
- Identified obesity patterns across patients
- Explored patient demographics and allergy data

## 📁 Files in This Repository
| File | Description |
|------|-------------|
| `patient_records.csv` | 1000 patient names & gender |
| `patient_details.csv` | Patient full details |
| `obesity_analysis.csv` | Obesity classification data |
| `gender_count.csv` | Gender distribution of patients |
| `allergy_analysis.csv` | Allergy patterns among patients |
